## 背景：TDS 水槽内 0ppm 現象

- **現象**: 小瓶（10mL）では 485ppm 程度の正常値、水槽内では 0ppm
- **原因**: 大容量の水が寄生静電容量・アンテナとして機能し、TDS の 3.95kHz AC 励起信号が GND へ分流（Shunting）
- **参照**: 2026-03 に受領した技術レポート（環境インピーダンス・回路論的分析）

### 暫定運用（アイソレータ導入まで）

水槽内では 0ppm のため、**1日1回ガラス瓶で測定**する。

- **手順**: 水槽から 10mL 程度を採取 → 瓶にプローブを浸す → `measure_tds_bottle.py` 実行
- **スクリプト**: `collector/scripts/measure_tds_bottle.py`（`.env` 変更・docker compose 不要）
- **DB 保存**: `--save` オプション（db コンテナが起動していること）
- **sensor_id**: `tds_bottle`（Grafana で連続測定と区別可能）

## タスク1: ガルバニック絶縁（DFR0504）導入

### 目的
TDS センサーと Raspberry Pi の GND を電気的に分離し、水槽の寄生容量への信号分流を阻止する。

### 使用モジュール
- **DFRobot Gravity: Analog Signal Isolator (DFR0504)**
- 内蔵 DC-DC でセンサー側に独立電源（5V）を供給
- アナログ信号を 1:1 で絶縁伝送

### ピン変更（v2.0 → v2.1 予定）

#### Raspberry Pi 側の変更

| 物理ピン | 現状(v2.0) | 変更後(v2.1) | 備考 |
|:--------:|------------|-------------|------|
| 17 | 3.3V → TDS/ADC | **2 または 4** | **5V** を DFR0504 電源入力に。DFR0504 は 5V 必要 |
| 9 | GND → 共通 | 変更なし | DFR0504 電源 GND |
| 3, 5 | I2C → MCP3424 | 変更なし | MCP3424 は Pi GND のまま |

**重要**: ラズパイの 3.3V では DFR0504 の DC-DC が動作しない可能性あり。**5V ピン（2 または 4）を使用**すること。

#### 配線の変更（概念図）

```
[現状 v2.0]
Pi 3.3V ──┬── MCP3424 VDD
          └── TDS VCC
Pi GND ───┬── MCP3424 VSS, CH1-
          └── TDS GND
TDS A ───────── MCP3424 CH1+

[変更後 v2.1]
Pi 5V ───────────── DFR0504 電源入力 (VCC)
Pi GND ──────────── DFR0504 電源 GND

DFR0504 絶縁側(A) ←── TDS センサー (VCC, GND, A)
  ※TDS はアイソレータの絶縁電源で駆動（Pi と GND 非共有）

DFR0504 出力側(B) ──→ MCP3424 CH1+, CH1-
  ※Signal → CH1+, GND → CH1-（疑似差動）
```

#### DFR0504 ピン確認（要実機確認）
- 左コネクタ (A+, A-): センサー側（絶縁領域）
- 右コネクタ: 電源入力 + ADC 出力側
- データシートで VCC/GND/SIG の対応を確認すること

### 注意事項
- TDS センサーの GND を Pi GND に**直結しない**（アイソレータの絶縁側 GND に接続）
- 5V 供給の電流容量: ラズパイ USB 電源から取得。DFR0504 + TDS で十分な余裕あり

## タスク2: ADC フロントエンド改善

### 目的
ノイズ耐性向上。アイソレータ導入後に実施。

### 変更内容

| 項目 | 現状 | 変更案 |
|------|------|--------|
| 差動入力 | CH1- を GND 直結 | アイソレータ出力 GND を CH1- に（疑似差動） |
| RC フィルタ | なし | CH1+ と CH1- 間に 1kΩ + 0.1µF の LPF |
| サンプリング | 240 SPS | 3.75 SPS（18bit） |

### ソフトウェア変更
- `gpio_tds.py` および `read_mcp3424_ch1.py` の MCP3424 設定バイト
- 0x80 → 0x9C 等（bit2-1 を 11 にして 3.75 SPS）

### ピン変更
なし（アイソレータ配線時に CH1- をアイソレータ GND に接続するだけ）

## タスク4: グラウンドプローブの注意

### 警告
